HTML

Now we need to add a set of html tags and you type it like this below. You need to put the html tag in because it is a container and holds 99% of our code.

To do the html first put a less than sign (<) then type "html" then put a greater than sign (>) for the opening tag. For the closing tag type a less than sign (<) then type "/html" then you type the greater than sign (>).

The html tag by itself doesn't do anything to your webpage see.
