Introduction to HTML


What is HTML


HTML stands for Hyper Text Markup Language. It is a Declarative language not a Computational such as Python, PHP, Javascript. It is used to announse what should exist on a website. You make a HTML file from saying anything you want then say .html .


HTML advantages


HTML is easy to learn, use and write (you will also need css and javascript to make the website more useful and beautiful).


HTML disadvantages


HTML by itself is very ugly (need to use css to do the styling). HTML has no animations (need to use javascript to do the animations).
